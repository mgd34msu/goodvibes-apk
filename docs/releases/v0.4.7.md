# GoodVibes Companion v0.4.7

Icon release for GoodVibes Companion.

## Highlights

- Adds the GoodVibes launcher icon from the checked-in `assets/branding/goodvibes-icon.png`
- Generates Android launcher icon PNGs for all current density buckets

## Behavior Notes

- App behavior is unchanged from v0.4.6
- The app still uses `@pellux/goodvibes-sdk` 0.33.23

## Setup

You will need:

- a running GoodVibes daemon compatible with SDK 0.33.23
- a reachable daemon URL such as `http://192.168.x.x:3210`
- daemon username/password credentials or a bearer token with the scopes you need

## Known Constraints

- No iOS build artifact in this release
- Phone must be able to reach the daemon URL directly
- Approval, session-write, and chat behavior still depend on the scopes granted by the daemon auth in use
